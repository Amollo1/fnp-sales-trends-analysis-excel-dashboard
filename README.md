# FNP Sales Trends Analysis Dashboard

## Project Overview

An end-to-end Sales Analytics Dashboard developed in Microsoft Excel for Ferns and Petals (FNP), a gifting company specializing in occasion-based gift deliveries.

The project transforms raw sales data into actionable business insights by analyzing revenue trends, customer purchasing behavior, product performance, delivery efficiency, and occasion-based sales patterns.

## Dashboard Preview

![Dashboard](https://github.com/Amollo1/fnp-sales-trends-analysis-excel-dashboard/blob/master/Dashboard_Iage.jpeg)

## Business Questions Addressed

* Total Revenue Analysis
* Monthly Sales Performance
* Top Products by Revenue
* Customer Spending Analysis
* Top 10 Cities by Orders
* Order Quantity vs Delivery Time
* Revenue by Occasion
* Product Popularity by Occasion

## Project Workflow

### Data Preparation

* Data Extraction and Cleaning using Power Query
* Data Transformation and Validation

### Data Modeling

* Built a relational model using Power Pivot
* Connected Customers, Orders, and Products tables

### Analysis

* Created Pivot Tables for KPI calculations and business insights

### Dashboard Development

* Designed an interactive dashboard with slicers, timelines, charts, and KPIs
